# Features

- **Advanced Invisibility Potion**: Brew a new type of invisibility potion that hides your identity.
- **Anonymous Kills**: When you kill another player while under the effect of the Advanced Invisibility Potion, your name will be displayed as "Unknown" in the death message.
- **Custom Recipe**: Brew the Advanced Invisibility Potion by combining a Phantom Membrane with a standard Potion of Invisibility (regular or long, drinkable or splash).

# How to Use

1. Download the latest version of the plugin.
2. Place the `AdvanceInvis-vX.X.X.jar` file in your server's `plugins` folder.
3. Start or restart your server. The plugin is now active for all players.
